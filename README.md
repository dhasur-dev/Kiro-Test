# Hello World Collection

A collection of "Hello World!" programs written in different programming languages.

## Brainfuck

**File:** `hello.bf`

[Brainfuck](https://en.wikipedia.org/wiki/Brainfuck) is an esoteric programming language created in 1993 by Urban Muller. It consists of only eight commands, each represented by a single character.

### Running

You can run the Brainfuck program using any Brainfuck interpreter. For example, using `beef`:

```sh
beef hello.bf
```

### Output

```
Hello World!
```

# Kiro-Test

A collection of "Hello, World!" programs in various languages.

## Programs

- `hello.bf` - Hello World in [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck)
- `hello.rs` - Hello World in [Rust](https://www.rust-lang.org/)

## Running

### Brainfuck

Use any Brainfuck interpreter to run `hello.bf`.

### Rust

```sh
rustc hello.rs -o hello && ./hello
```
